LaaS Dashboard

v6.2  ·  Now Live for the Team

Dear Team,

I am pleased to announce that the LaaS (Lab as a Service) Dashboard v6.2 is now fully operational and ready for daily use by the entire SDX Infrastructure & Data Centre team. This unified portal consolidates eight enterprise infrastructure platforms into a single browser-based interface — eliminating the need to log into individual platform consoles.

https://<server-ip>
HTTPS · Port 443 · Use your AD credentials (sdxtest.local) to sign in

🔗 Platforms Integrated

VMware vCenter — VMs, hosts, snapshots, cloning, vMotion
Red Hat OpenShift — Clusters, nodes, pods, operators, RHOAI
Nutanix Prism Central — Multi-site DC/DR, AHV VMs, storage, alerts
Ansible AAP — Job templates, inventories, live execution
Amazon Web Services — EC2, S3, RDS, VPCs, Cost Explorer
Microsoft Hyper-V — VMs, checkpoints, CPU/RAM via WinRM
Active Directory & DNS — Users, groups, DNS records via LDAP
IPAM (SolarWinds) — Subnets, IP utilisation, IP search

⚡ Key Capabilities

Full VM lifecycle — power, clone, migrate, reconfigure, snapshots
OpenShift node cordon/drain, pod logs, operator health
Self-service VM Request Wizard (VMware / OCP / Nutanix)
Chargeback — monthly costs in ₹ INR and $ USD per project
Universal Search — find any VM, host, pod, contact instantly
LaaS AI Assistant — ask natural-language infrastructure queries
Asset Inventory with inline editing and CSV export
Full Audit Log — every portal action tracked and exportable

📊 New in v6.2 — Insights & Analytics Engine NEW

  • Health Scorecard — Real-time RAG (Red/Amber/Green) status across all 8 platforms with trend arrows
  • Change Detection — Automatic day-over-day drift alerts for VM count, host count, pod count, IP utilisation
  • Capacity & Cost — Storage/compute exhaustion predictions with monthly chargeback in ₹ and $
  • Executive Summary — One-page PDF-ready overview with aggregate KPIs and platform health matrix
  • Historical Trending — Interactive charts for 7, 14, and 30-day trends across all metrics
  • Capacity Forecasting — Linear regression predictions for storage, compute, and network exhaustion dates
  • Daily Snapshots — Automated nightly collector (11 PM) captures data from all platforms into PostgreSQL

The dashboard supports three themes (Dark, Light, Wipro Wings) and features a voiceover walkthrough accessible via the speaker button next to the About LaaS section. Role-based access (Admin, Operator, Viewer, Requester) ensures each team member sees exactly what they need.
